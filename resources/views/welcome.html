Welcome to Onyx! 🎉

You're running a Laravel-inspired framework built with Go.

✨ Features

  • 🛣️ Expressive routing with parameter binding
  • 🔧 Powerful middleware system
  • 📦 Dependency injection container
  • 🗄️ Database ORM and query builder
  • ⚙️ Configuration management
  • 🎨 Template engine with layouts
  • ✅ Request validation
  • 🔨 CLI tools for scaffolding

🚀 Quick Start

app := framework.New()

app.Get("/", func(c *framework.Context) error {
    return c.String(200, "Hello Onyx!")
})

app.Start(":8080")
